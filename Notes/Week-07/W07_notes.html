Call Methods

Method syntax: myfunction.call(parameter)
This method can be used to set the value of this inside a function.
Example:
Function myfunction(){
return “Hi, my name is ${this.name}”;
}
const jacob = { name: ‘Jacob’ };
const rani = { name: ‘Rani’ };
myfunction.call(Jacob);
output: “Hi, my name is Jacob”
myfunction.call(rani);
output: “Hi, my name is Rani”

***to make a function more generalized***
Function myfunction(greeting=”Hi”){
return “${ greeting }, my name is ${ this.name}”;
myfunction.call(jacob, ‘How do you do’);
output: “How do you do, my name is Jacob”
myfunction.call(rani);
output: “Hi, my name is rani”
*************************************
In myfunction(greeting=”Hi”), greeting=”Hi” is setting up a default greeting word in case there is no specific input is passed to the function.
**** If call function is not referring an object as this in its body, it can be used by providing null the its first argument.
Example:
Square.call(null, 4)
Output: 16
++++ apply() method will work in the same way. However apply() method will receive argument as an array even those there is only one argument.
Example:
Square.apply(null, [4])
Output: 16

Memoization

Memoizaiton is same as caching.
Function can store caches for better response time. Especially when function takes time to calculate or compute a return value.
Example:
Function square(x){
Square.cache = square.cache || {};
If (!square.cache[x]) {
Square.cache[x] = x*x;
}
return square.cache[x]
}
Square(3); output: 9
Square(-11); output: 121
Square.cache; output: {“3”: 9, “-11”: 121}

Immediately Invoked Function Expressions

IIFE is an anonymous function that is invoked as soon as it is defined.
Example:
(function(){
Const temp = “World”;
Console.log(“Hello ${temp}”);
})();
Output: “Hello World”
