CH 8 CSS3 Transforms and Transitions

        -Transforms - manipulate an element's appearance. 
            -Translation - allow you to move elements left, right, up, or down. Move elements without impacting the flow of the document.
            -Scaling - defined factors horizontally then vertically. The center stays in the same place as its dimensions change.
                       No impact on the document flow. 
            -Rotation - rotates an element around the point of origin by a specified angle value. The point of origin is the element's center.
                        positive degrees move clockwise, and negative moves it counter clockwise. 
            -Skew - specifies a skew along the x and y axes. If the second parameter is omitted, the skew will only occur on the x axis. 
            -Changing the Origin of the Transform - changes the center of the object.
            - The order of transform functions does matter: if you rotate before translating, your translate direction will be on the rotated axis.
        -Transitions allow the values of CSS properties to change over time, essentially providing simple animations. 
            -Transitions allow the values of CSS properties to change over time, essentially providing simple animations.  
            -The transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state.  
            -The transition-timing-function lets you control the pace of the transition in even more granular detail.
            -The transition properties allow for multiple transitions in one call.
        -Animations 
            -CSS animations, unlike transitions, allow you to control each step of an animation via keyframes.  
            -A keyframe is a snapshot that defines a starting or end point of any smooth transition.   
            -The lack of powerful processors on many mobile devices makes CSS animations a great alternative to weighty, CPU-intensive JavaScript animation.
            -The transition properties allow for multiple transitions in one call.
