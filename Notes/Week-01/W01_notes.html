Doing Stuff With Web Things

    Required Tools: 
        - Laptop or desktop computer
        - latest version of Chrome
    
    Steps to Success:
        - Think => find out what you don't know and learn it by playing with it (sandboxing).
        - Design => use what you have just learned to layout a solution to the problem.
        - Test => create a series of expectations for behavior that will indicate if you have been successful.
        - Create => write the code!
